# 🎓 Educai

**Educai**, modern yazılım mimarilerini oyunlaştırma prensipleriyle harmanlayan, öğrencilerin akademik gelişimlerini dinamik ve rekabetçi bir ortamda sürdürmelerini sağlayan hibrit bir öğrenme platformudur. 

Türkiye ortaöğretim müfredatını temel alan geniş soru havuzu, gerçek zamanlı düello mekanikleri ve kişiselleştirilmiş AI asistanı ile eğitimde yeni nesil bir deneyim sunar.

---

## 🚀 Proje Vizyonu
Educai, sadece bir soru çözme uygulaması değil; kullanıcının tercihlerine, öğrenme hızına ve hedeflerine göre şekillenen bir **Akıllı Eğitim Asistanı**dır. Real-time iletişim katmanıyla öğrencileri birbiriyle yarıştırırken, RPG öğeleriyle (karakter evrimi, XP sistemi) öğrenme sürecini sürdürülebilir bir motivasyon döngüsüne dönüştürür.

## 🛠 Teknik Stack
Bu proje, web ve mobil dünyasının en güçlü teknolojilerini merkezi bir .NET API üzerinden birleştirir:

* **Backend:** ASP.NET Core 10.0 (C#), SignalR (Real-time Engine), PostgreSQL.
* **Web (Dashboard):** React & Tailwind CSS.
* **Mobile (iOS):** Swift & SwiftUI (Combine Framework ile Fonksiyonel Programlama yaklaşımı).
* **AI Integration:** OpenAI API (Akıllı Profil ve Soru Analizi) - (Proje canlıya alınana kadar Mock).

## 💡 Temel Özellikler
* **Real-time Matchmaking:** SignalR altyapısı ile anlık eşleşme ve canlı bilgi yarışması modları.
* **AI-Powered Personalization:** Kullanıcı tercihlerine göre şekillenen müfredat odaklı soru akışı.
* **Character Evolution:** Seviye atladıkça gelişen karakter tasarımları ve XP sistemi.
* **Cross-Platform:** Web ve Mobil platformlar arasında senkronize ilerleme.
