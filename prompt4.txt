Görev 4: XP Hesaplama — DifficultyWeight Çarpanı

Düello sonuçlarına göre DifficultyWeight çarpanı kullanarak 
XP hesaplama mantığını kur.

Mevcut mimari:
- Servisler sealed class
- DTO'lar immutable record
- Direkt EducaiDbContext kullanılıyor
- Guid.CreateVersion7() ile ID üretiliyor
- AsNoTracking() read sorgularında kullanılıyor
- Challenge entity'sinde DifficultyWeight (XP çarpanı) var
- ChallengeMatch entity'sinde Player1, Player2, WinnerId, MatchStatus var
- LeaderboardService'de mevcut XP = 100 × Challenge.DifficultyWeight mantığı var
- User entity'sinde Level, TotalXP, AntigravityMultiplier var

Yapılacaklar:

1. Application katmanında:
   - XPRewardDto immutable record oluştur
   - IXPService interface'i tanımla
   - Metotlar:
     Task<XPRewardDto> CalculateMatchXPAsync(Guid matchId)
     Task ApplyXPToWinnerAsync(Guid matchId)

2. Infrastructure katmanında:
   - sealed XPService : IXPService implementasyonu
   - XP hesaplama: 100 × Challenge.DifficultyWeight × AntigravityMultiplier
   - Sadece WinnerId olan ve MatchStatus.Completed maçlara uygula
   - Kazananın TotalXP'sini güncelle
   - Level hesaplama: her 1000 XP'de bir level atla

3. API katmanında:
   - XPController oluştur
   - POST /api/xp/calculate/{matchId} endpoint'i
   - POST /api/xp/apply/{matchId} endpoint'i

Kurallar:
- Tüm metotlar async/Task yapısında olsun
- Mevcut sealed servis ve immutable record yapısına uy
- Dependency Injection kaydını yap
- XML yorum satırları ekle
